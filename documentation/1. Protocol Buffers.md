# Protocol Buffers



## Overview

Protocol Buffers 는 구조화된 데이터를 언어 중립적, 플랫폼 중립적으로 확장 가능한 매커니짐으로 직렬화, 역직렬화 하기 위한 매커니즘이다. 

JSON가 유사하지만, 데이터가 더 작고 빠르며 여러 언어에 바인딩 할 수 있다. 

.proto 라는 확장자를 가진 정의 파일을 만들고, 언어별 protoc 컴파일러를 통해 원하는 언어로 컴파일 한다.
